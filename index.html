예적금 만기 계산기
예금과 적금의 만기 수령액을 계산합니다. 단리·복리, 세금 3종을 선택할 수 있습니다.
단리/월복리 이자소득세 도넛 차트

대출 이자 계산기
주택담보·신용대출의 월 상환금과 총 이자를 원리금균등·원금균등·만기일시 3가지 방식으로 계산합니다.
원리금균등 원금균등 거치기간 상환 스케줄
→
📈
복리 수익률 계산기
투자 원금과 기대 수익률로 복리 효과를 계산합니다. 월 추가투자, 72의 법칙, 연도별 성장 그래프를 제공합니다.
복리 vs 단리 월 추가투자 72의 법칙
→

전세/월세 전환 계산기
전세금을 월세로, 월세를 전세로 전환할 때 적정 금액을 계산합니다. 법정 전월세 전환율(연 6%) 기준.
전세→월세 월세→전세 보증금 조정
→

연금 수령액 계산기
국민연금 예상 수령액, 퇴직연금, 개인연금(연금저축·IRP)을 계산하고 노후 필요 생활비와 비교합니다.
국민연금 퇴직연금 개인연금 통합 분석

세후 이자 계산기
이자소득세(15.4%), 세금우대(9.9%), 비과세 3가지 유형을 동시에 비교해 세후 실수령 이자를 계산합니다.
일반 15.4% 세금우대 9.9% 비과세 0%

물가상승률 계산기

미래 가치 구매력 감소 실질 수익률
→
💱
환율 계산기
원화와 달러·엔화·유로·위안화·파운드를 환산합니다. 환전 수수료를 포함한 실제 환전 금액도 계산합니다.
USD·JPY·EUR CNY·GBP·VND 수수료 계산
→

실시간 즉시 계산
입력하는 즉시 결과가 계산됩니다. 서버 요청 없이 브라우저에서 바로 처리합니다.
📊
시각적 차트 제공
도넛 차트, 라인 차트, 막대 그래프로 결과를 시각화하여 이해하기 쉽습니다.
📱
모바일 최적화
PC와 스마트폰 모두에서 최적의 사용 경험을 제공합니다.
🔒
개인정보 수집 없음
모든 계산은 브라우저에서 처리되며 어떤 개인 정보도 서버로 전송되지 않습니다.
💯
완전 무료
